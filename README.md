# Android Notice Board App

###  프로젝트 개요

본 프로젝트는 Java를 사용해 개발한 Android 기반 게시판 애플리케이션입니다.

사용자 로그인부터 게시글 작성, 조회, 수정, 삭제까지
게시판의 기본적인 기능을 구현
Android 앱과 서버 간의 데이터 통신을 통해
실제로 동작하는 게시판 서비스를 구성했습니다.



###  주요 기능

##### 사용자 인증
 회원가입 및 로그인 기능
 아이디 중복 확인
 아이디 / 비밀번호 찾기

##### 게시판 기능
 게시글 목록 조회
 게시글 작성, 수정, 삭제
 게시글 상세 조회

 ##### 서버 통신
 Android 앱과 서버 간의 데이터 통신
 서버에서 전달받은 게시글 데이터를 화면에 표시


 ### 구현 특징

 Android 앱에서 서버로 요청을 보내 사용자 인증 및 게시글 관리 처리
 서버 응답 데이터를 JSON 형식으로 받음
 게시글 목록을 구성해 입력에 따라 화면 업데이트



 ### 사용 기술

 Android (Java)
 PHP
 MySQL
 JSON
